Panel de gestión agrícola con Frontend en Vite + React. Estructura inicial con carpeta `Frontend/` y una carpeta `Backend/` (a definir).

## Estructura

- `Frontend/`: aplicación React (Vite)
- `Backend/`: placeholder para API/servicios

## Requisitos

- Node.js 18+
- PNPM o NPM

## Ejecutar Frontend (desarrollo)

```pwsh
Push-Location "Frontend"
# instala dependencias
npm install
# levanta el servidor de desarrollo
npm run dev
Pop-Location
```

## Build Frontend (producción)

```pwsh
Push-Location "Frontend"
npm run build
Pop-Location
```

## Scripts útiles

- `Frontend/package.json`: contiene scripts `dev`, `build`, `preview`.

## Contribución

1. Crea una rama desde `main`
2. Commits con mensajes descriptivos
3. Pull Request con cambios y notas
